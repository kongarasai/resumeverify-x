# ResumeVerify X™ — Universal AI Career, Learning, Placement & Hiring Operating System

## What's Included

### 6 Complete Role Dashboards
1. **Candidate/Student** — Full career OS
2. **Teacher** — AI teaching tools, assignments, analytics  
3. **Mentor** — Batch management, live interviews, AI coaching
4. **HR/Recruiter** — Candidate pipeline, live interview room, AI copilot
5. **Placement Officer** — Drives, analytics, eligibility engine
6. **Super Admin** — Fraud monitoring, university management, subscriptions

## Tech Stack
- Frontend: Next.js + TypeScript + Tailwind CSS + ShadCN UI + Framer Motion
- Mobile: React Native Expo + NativeWind
- Backend: NestJS + Prisma ORM
- Database: PostgreSQL
- Realtime: Socket.IO + Redis Pub/Sub
- AI: OpenAI API + LangChain + Whisper
- Video: WebRTC + LiveKit + Agora
- Compiler: Monaco Editor + Judge0 + Docker
- Storage: AWS S3 + Cloudinary
